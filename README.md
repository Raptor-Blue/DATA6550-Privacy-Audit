# DATA6550-Privacy-Audit
Group project for DATA 6550 on Data Privacy

## Structure
- `privacy_audit_report.pdf`: Final project report for presentation.
- `Agreements/`: Directory containing individual user agreement documents.
- `Analysis/`: Directory containing individual analyses documents.
  - `Forte/`: Documents for Kristen Forte.
  - `Segura/`: Documents for Sebastian Segura.
  - `Goswani/`: Documents for Tanviben Goswami.
  - `Lingala/`: Documents for Keerthana Lingala.
- `Comparison/`: Directory containing the comparison matrix.
  - `regulatory_matrix`: Excel matrix comparing all privacy agreements.
- `Collaboration/`: AI-generated summaries of group discussions.
  - `WeekA.docx`: Summary of Week A discussions.
  - `WeekB.docx`: Summary of Week B discussions.
